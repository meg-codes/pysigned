# Installation

`pysigned` requires **Python 3.13+**.

## uv

```sh
uv add pysigned
```

To install into the current environment without adding it to a project:

```sh
uv pip install pysigned
```

## pip

```sh
pip install pysigned
```

## Verifying the install

```python
import pysigned

print(pysigned.__all__)
```
